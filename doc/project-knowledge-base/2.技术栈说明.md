# 技术栈说明

## 1. 技术栈总览

> OpenPROM 采用纯 Python 后端 + 纯静态 SPA 前端的一体化架构。所有智能能力均通过远程 OpenAI 兼容 API 调用；规则引擎（格律、平仄、押韵）封装为 LLM Tool，供模型自修正使用。部署侧使用 Docker Compose 提供 API + Redis + Prometheus + Grafana 完整栈。

## 5. 开发环境要求

### 5.1 环境依赖

| 依赖     | 最低版本 | 推荐版本 | 安装方式       |
| -------- | -------- | -------- | -------------- |
| Python   | 3.9      | 3.11+    | pyenv / 官方包 |
| pip      | 随 Python | 最新    | 自带           |
| Docker   | 24.x     | 最新     | 官方安装包     |
| Docker Compose | 2.x | 最新   | Docker Desktop |

### 5.2 推荐 IDE 与插件

| IDE / 编辑器 | 推荐插件                          |
| ------------ | --------------------------------- |
| VS Code      | Python, Ruff, Markdown All in One |

### 5.3 本地启动步骤

```bash
# 1. 进入项目根（包含 pyproject.toml 的目录）
cd openprom

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env，至少填入 OPENPROM_API_KEY

# 4. 初始化数据库
python -c "from openprom.infrastructure.database import get_db_manager; get_db_manager().create_tables()"

# 5. 启动服务
python -m openprom.api
```

# 术语定义

## 1. 值域定义（枚举）

### 1.1 对联评级（COUPLET_GRADE）

| 值       | 说明                       |
| -------- | -------------------------- |
| 优秀     | 总分 ≥ 85，意境与对仗俱佳  |
| 良好     | 总分 70–84，整体协调       |
| 及格     | 总分 60–69，基本合规       |
| 不合格   | 总分 < 60，需大幅调整      |

---

### 1.2 格律类型（METER_TYPE）

| 值       | 说明                       |
| -------- | -------------------------- |
| shi      | 诗 / 律诗 / 绝句           |
| ci       | 词牌                       |
| couplet  | 对联                       |

---

### 1.3 平仄标记（TONE_VALUE）

| 值  | 说明   |
| --- | ------ |
| 1   | 平声   |
| -1  | 仄声   |
| 0   | 中声（可平可仄） |

---

### 1.4 任务状态（TASK_STATUS）

| 值       | 说明                       |
| -------- | -------------------------- |
| success  | 任务成功完成               |
| fail     | 任务执行失败               |

---

### 1.5 主题模式（THEME_MODE）

| 值       | 说明                       |
| -------- | -------------------------- |
| light    | 浅色主题（宣纸米白）       |
| dark     | 深色主题（墨黑）           |

---

## 2. 业务术语

| 术语              | 定义                                                         |
| ----------------- | ------------------------------------------------------------ |
| 对联              | 由上下两句组成、字数相等、结构对称、平仄协调的汉语文体。     |
| 律诗              | 讲究平仄、对仗、押韵的格律诗，常见五绝、七绝、五律、七律。   |
| 平仄              | 汉语声调的分类：平声（阴平、阳平）为平，上声、去声、入声为仄。 |
| 对仗              | 上下联在词性、结构、意义上相互对应。                         |
| 押韵              | 诗句特定位置使用同一韵部的字。                               |
| 马鞍工程          | 多层 LLM 输出质量控制机制，包含形式 / 技法 / 艺术 / 印象四层。 |
| Agentic Loop      | LLM 通过调用工具进行多轮自修正的循环。                       |
| RAG               | Retrieval-Augmented Generation，检索增强生成。               |
| Embedding         | 将文本映射为稠密向量的过程，用于语义检索。                   |
| Rerank            | 对初步召回结果进行精排序。                                   |
| ChromaDB          | 项目使用的向量数据库。                                       |
| Skill             | 针对特定任务类型（生成对联、评鉴等）的检索策略。             |
| Session ID        | 前端生成的匿名会话标识，用于历史与反馈关联。                 |
| Task Trace        | Agent 执行过程的持久化记录，包含 LLM/工具/RAG 调用次数。     |

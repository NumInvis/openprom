## 3. 表结构详情

### 3.1 couplet_analyses — 对联分析记录

| 字段名             | 类型       | 可空 | 默认值   | 说明                     |
| ------------------ | ---------- | ---- | -------- | ------------------------ |
| id                 | Integer    | 否   | 自增     | 主键                     |
| upper              | String(200)| 否   | -        | 上联                     |
| lower              | String(200)| 否   | -        | 下联                     |
| formal_score       | Float      | 是   | 0.0      | 形式合规分               |
| technique_score    | Float      | 是   | 0.0      | 对仗技术分               |
| artistic_score     | Float      | 是   | 0.0      | 艺术表现分               |
| impression_score   | Float      | 是   | 0.0      | AI 印象分                |
| total_score        | Float      | 是   | 0.0      | 总分                     |
| grade              | String(20) | 是   | ""       | 评级                     |
| pingze_score       | Float      | 是   | 0.0      | 平仄协调分               |
| llm_evaluation     | JSON       | 是   | {}       | LLM 原始评价             |
| warnings           | JSON       | 是   | []       | 警告列表                 |
| comments           | JSON       | 是   | {}       | 评语（技法/艺术/印象）   |
| session_id         | String(64) | 是   | -        | 匿名会话 ID              |
| request_id         | String(64) | 是   | -        | 请求追踪 ID              |
| is_public          | Integer    | 是   | 0        | 是否公开                 |
| favorite           | Integer    | 是   | 0        | 是否收藏                 |
| tags               | JSON       | 是   | []       | 标签                     |
| created_at         | DateTime   | 是   | now      | 创建时间                 |
| updated_at         | DateTime   | 是   | now      | 更新时间                 |

**索引**：`idx_scores` (total_score, grade)、`idx_created` (created_at)、`idx_session` (session_id)

## 4. 数据流说明

1. **评鉴流程**：用户提交对联 → 形式分析 → LLM 技法/艺术/印象 → 马鞍校准 → 写入 `couplet_analyses`。
2. **生成流程**：用户提交主题 → LLM Tool Loop（检索/格律检测）→ 输出结果 → 可选写入 `task_trace`。
3. **格律检测**：用户提交文本 → `meter`/`pingze` 引擎 → 返回结果并写入 `meter_checks`。
4. **反馈闭环**：用户对评鉴结果提交反馈 → 写入 `user_feedbacks`。
